# File Encryption & Decryption in C

## 📌 Description
This project demonstrates simple file encryption and decryption using ASCII manipulation in C.

## 🔐 Algorithm Used
Encryption:
ch = ch + 100

Decryption:
ch = ch - 100

## 📁 Files
- encrypt.c → Encrypts file content
- decrypt.c → Decrypts file content
- input.txt → Sample file

## ▶️ How to Run

### Compile
gcc encrypt.c -o encrypt
gcc decrypt.c -o decrypt

### Run
encrypt.exe
decrypt.exe

## 📌 Example

Input:
hello this is my C project

Encrypted:
(unreadable characters)

Decrypted:
hello this is my C project
